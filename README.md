# ELEPOKE ポケモン選出最適化アプリ API
https://elepoke.site

## API 利用方法 (ローカル環境)
```bash
git clone https://github.com/Alesion30/elepoke_api.git
cd elepoke_api
python -m venv virt
. virt/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

vscodeを起動 -> デバッグ -> Python: Flaskを実行

もしくは

vscodeを起動 -> F5を押す
